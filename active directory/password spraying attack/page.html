wordlister
https://github.com/4n4nk3/Wordlister


1. make a words.txt (possible words)

2. combine the words

python3 wordlister.py --input words.txt --perm 3 --min 8 --max 32

3. add numbers

python3 wordlister.py --input words.txt --perm 3 --min 8 --max 32 --append '123'

4. leet commbination

python3 wordlister.py --input words.txt --perm 3 --min 8 --max 32 --leet
